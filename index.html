如何升級?

升級條件是依據各冒險者對於「中央大學天文所Python使用者討論會」GitHub專案 的 commit 次數。等級1升到等級2需要 commit 4 次，等級2之後，每個等級間距所需的 commit 次數為2的 n 次方，n 為目前級數, 例如: 等級2到等級3需要再 commit 4 次、 3到4需要再 commit 8 次、4到5需要再 commit 16 次 ...

關於如何commit，請參閱此文件「如何使用Git/GitHub協作討論會的筆記」的說明。

如何解開技能?
  • 開源天文技能 (技能名稱 : 解開條件)
    1. 初等 : 針對「中央大學天文所Python使用者討論會」GitHub專案 commit 5次以上。
    2. 中等 : 先解開上一個技能，並且參與過一次討論會成員辦的天文駭客松(hackathon)。
    3. 高等 : 先解開上一個技能，並且在「Astrohackers in Taiwan」GitHub組織中主導過一個新專案。
  • 戰鬥技能 (技能名稱 : 解開條件)
    1. 初等 : 到CheckiO 或 HackerRank 這兩個Online Judge線上解題平台上找題目來做，並分享解題經驗給一同冒險的夥伴們3次以上。
    2. 中等 : 先解開上一個技能，並且分享Python物理數值分析相關題目(求根、線性代數、微分方程、數值微積分、統計...)的解題經驗給一同冒險的夥伴們5次以上。
    3. 高等 : 先解開上一個技能，並且自己出10題天文相關的Pyrhon題目給一同冒險的夥伴們解。
  • 天問技能 (技能名稱 : 解開條件)
    1. 初等 : 在「中央大學天文所Python使用者討論會」GitHub專案的issues頁面 上提交5次以上關於討論會筆記的改進提案。

  • 角色技能

    依據所屬研究領域，每位冒險者會被賦與一個角色，藉由分享天文研究相關Python套件的次數以解開所屬角色不同層級的技能。(以下所列次數是不含前一個技能的所需次數)

    • 星際信使 (太陽系) : 初級 (1次) -> 中級 (3次) -> 高級 (5次)

如何開始與我們一同冒險?

請完成以下加入冒險的步驟:
  1. 註冊一個GitHub帳號
  2. 到「中央大學天文所Python使用者討論會」GitHub專案的issues頁面 ，按"New issue"新增一個issue，標題請填"申請成為此專案的合作者", 內容填上你的GitHub帳號名稱後按"Submit new issue"送出issue。
  3. 等專案管理者把你加入到此專案的合作者名單後，你的GitHub帳號名稱就會列在"冒險者們"之中，此時你就完成加入冒險的步驟囉! 然後查閱關於如何升級和解開技能的規則，開始冒險吧!
